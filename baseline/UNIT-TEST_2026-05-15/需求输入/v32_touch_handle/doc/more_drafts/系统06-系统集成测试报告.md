## 2. 测试概述

### 2.1 测试目标
验证门把手触摸功能在真实系统环境下的集成表现，重点包括：
- 用户触摸 → 开锁信号输出 → BCM 响应的端到端流程
- GPIO 硬线接口与 LIN 通信接口的协同工作
- 低功耗模式（睡眠/唤醒）与车身控制器的状态同步
- 在振动、高低温、电源波动等环境下的鲁棒性

### 2.2 测试范围
- 核心功能集成（5项）
- LIN 通信集成（3项）
- 电源与低功耗集成（2项）
- 异常与环境适应性（5项）

### 2.3 测试环境

| 类别 | 设备/工具 | 说明 |
|------|-----------|------|
| **硬件** | 门把手总成 x3 | 量产等效样件（Rev C2） |
| | BCM 仿真器 | 模拟车身控制逻辑 |
| | Vector VN1630 | LIN 主机 |
| | Keysight DSOX1204G | 示波器，监测 `UNLOCK_SIGNAL` |
| **软件** | CANoe.LIN | LIN 诊断与 OTA 仿真 |

| | Jira | 缺陷跟踪 |
